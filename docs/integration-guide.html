🛠️ Métodos de Integración

📖 GitBook

Usa la sintaxis de embed de GitBook:

{% embed url="https://chusitooxduwu.github.io/graficsIP/visualizations/nivel2/math-module-explorer.html" %} Explorador interactivo del módulo math de Python {% endembed %}

🌐 HTML/Iframe

Para sitios web, Moodle, Canvas u otras plataformas:

<iframe src="https://chusitooxduwu.github.io/graficsIP/visualizations/nivel2/math-module-explorer.html" width="100%" height="600px" frameborder="0" allowfullscreen> </iframe>

📱 Responsive

Para dispositivos móviles, usa estas dimensiones:

<iframe src="https://chusitooxduwu.github.io/graficsIP/visualizations/nivel1/append-demo.html" width="100%" height="500px" frameborder="0" allowfullscreen style="border: 2px solid #007acc; border-radius: 10px;"> </iframe>

⚙️ Configuraciones Recomendadas

  • Ancho: 100% (responsive)
  • Alto: 500-600px según contenido
  • Borde: Opcional, mejora la apariencia
  • allowfullscreen: Recomendado para mejor experiencia
